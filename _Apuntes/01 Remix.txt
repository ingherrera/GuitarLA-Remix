"Remix Run" o solamente "Remix"es una alternativa a Next JS para crear sitios y aplicaciones en React que se ejecuten tanto en el cliente como en el servidor.

"Remix" segun sus creadores es:
*) Un compilador, 
*) Un administrador de peticiones HTTP, 
*) Un framework para el servidor y 
*) un framework para el cliente(osea el navegador)

Como esta hecho por los creadores de "Ract Router DOM" maneja "acciones" y "loaders"

Ventajas de remix

*) Cero configuraci√≥n(pr√°cticamente a la hora de instalarlo ya viene listo para producci√≥n)

*) Gran performance y optimizado para SEO.

*) Routing  incluido

*) Manejo de  peticiones HTTPS similar a "React Router DOM"

*) El c√≥digo sigue siendo React(componentes, hooks)
